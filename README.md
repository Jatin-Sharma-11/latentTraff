# LTFNet: Latent Traffic Field Networks

LTFNet is a neural PDE-based framework for spatio-temporal traffic forecasting. 
It models traffic as a continuous latent field, combining graph diffusion, 
nonlinear neural dynamics, and state-space temporal refinement.

The model achieves strong accuracy–efficiency trade-offs across real-world 
benchmarks, enabling scalable and stable long-horizon traffic prediction.
